What is Ember

ember.js is a framework for creating ambitious web applications
There are a lot of JavaScript libraries available, and most are really good at providing the traditional DOM-centric interactions that your typical websites need. But when it's time to build a manageable code base for a single-page app, that's where a whole suite of new frameworks come in to smooth things out.

Full stack

  • data-binding
  • event routing
  • state management
multiple libraries useful for web development into a single cohesive software stack for web developers to use. build your own gmail without worrying about jquery, ajax calls and browser compatibility.

Feel at home

  • MVC driven (though not webMVC)
  • client-side templates
  • Convention over configuration
